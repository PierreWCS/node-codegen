# API

Node server with ApolloServer, TypeScript, MongoDB native driver and Codegen

## Install dependancies

```
cd api && yarn
```

Generate types from schema

```
yarn codegen
```

Run the project locally

```
yarn dev
```

## DB connection

Define .env file to connect to the database
